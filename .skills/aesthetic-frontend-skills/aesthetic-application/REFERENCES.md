# References — Aesthetic Application

## Design Token Specifications

- **W3C Design Tokens Community Group (DTCG) Format Specification** — [https://design-tokens.github.io/community-group/format/](https://design-tokens.github.io/community-group/format/) — Canonical specification for the `$type` / `$value` JSON token format used in this skill's token output.

- **Theo (Token Pipeline)** — [https://github.com/theo-tokens/theo](https://github.com/theo-tokens/theo) — Tool for transforming W3C DTCG tokens to platform-specific formats (CSS custom properties, Sass variables, iOS Swift, Android XML). Useful for consuming the JSON output of this skill.

- **Style Dictionary** — [https://amzn.github.io/style-dictionary/](https://amzn.github.io/style-dictionary/) — Amazon's design token build system. Also supports DTCG format as input. An alternative to Theo for multi-platform token distribution.

## Accessibility

- **WCAG 2.1 Success Criterion 1.4.3: Contrast (Minimum)** — [https://www.w3.org/TR/WCAG21/#contrast-minimum](https://www.w3.org/TR/WCAG21/#contrast-minimum) — 4.5:1 for normal text, 3:1 for large text and UI components. Referenced in the Flags section of all generated specs.

- **WebAIM Contrast Checker** — [https://webaim.org/resources/contrastchecker/](https://webaim.org/resources/contrastchecker/) — Quick tool for verifying foreground/background contrast ratios against WCAG AA/AAA thresholds.

## Typography

- **Google Fonts** — [https://fonts.google.com/](https://fonts.google.com/) — Free, web-native fonts. Search by aesthetic period, classification, or visual characteristics. Primary source for font family recommendations in tokens.

- **Fonts In Use** — [https://fontsinuse.com/](https://fontsinuse.com/) — Searchable archive of typography in real-world design. Useful for verifying that a typeface actually appears in the target aesthetic's historical context.

- **Bringhurst, Robert** — *The Elements of Typographic Style* (4th ed., Hartley & Marks, 2012) — Authoritative reference for typeface classification, type history, and the relationship between typography and aesthetic period.

## Color Theory

- **Itten, Johannes** — *The Art of Color* (Van Nostrand Reinhold, 1973) — 7-color harmony types. Referenced when constructing palette relationships (complementary, triadic, split-complementary).

- **Fairchild, Mark D.** — *Color Appearance Models* (3rd ed., Wiley, 2013) — Perceptual color science. Referenced when assessing luminance, perceived warmth/coolness, and simultaneous contrast effects in generated palettes.
