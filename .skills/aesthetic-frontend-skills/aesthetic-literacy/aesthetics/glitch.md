**Palette**: JPEG artifact smears; signal-disruption green on black; unexpected magenta/cyan bleeding

**Type**: Corrupted text; misaligned channels; pixel displacement

**Texture**: Digital artifact, bit-error noise, pixel sorting, data-bend corruption

**Shape**: Fragmented rectangles; channel-shift distortion; no clean edges

**Motion**: Stuttering loop, frame-bleeding

**Spatial**: No conventional structure; planned chaos

**Cultural markers**: Error messages repurposed as art, "corrupted file" aesthetic, code-as-art

**Non-negotiables**: visual corruption artifacts + channel-shift distortion

**Subsets**: *Databending* — opening a file in software designed for a different format (e.g., a JPEG in Audacity as raw audio, applying effects, re-saving); produces smeared, streaked distortion that follows the logic of the wrong medium. *Datamoshing* — video-specific; deleting or duplicating codec I-frames and P-frames so motion vectors from one clip bleed into another clip's pixel content; the signature "skin melting off" or "reality dissolving" effect. Databending and datamoshing are techniques and their resulting aesthetics — subsets of glitch, not synonyms.

**Connotation**: System failure as expressive energy; unstable, critical, and cybernetic rather than smooth or trustworthy.
