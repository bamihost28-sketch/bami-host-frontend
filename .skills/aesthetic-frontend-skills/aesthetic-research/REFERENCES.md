# References — Aesthetic Research

## Visual Research Methodology

- Rose, G. (2016). *Visual Methodologies: An Introduction to Researching with Visual Materials* (4th ed.). SAGE. → Research methods for systematic visual analysis; informs the image annotation and synthesis workflow.

- Banks, M. (2007). *Using Visual Data in Qualitative Research*. SAGE. → Guidance on coding and categorizing visual material; relevant to the 7-dimension annotation process.

## Visual Reference Sources

- **Pinterest** — [https://www.pinterest.com](https://www.pinterest.com) — Strong source for mood boards and aesthetic imagery. Search URL pattern: `https://www.pinterest.com/search/pins/?q=<encoded-query>&rs=typed`

- **Dribbble** — [https://dribbble.com/search/](https://dribbble.com/search/) — UI-focused; high signal for digital UI aesthetics specifically.

- **Behance** — [https://www.behance.net/search/projects](https://www.behance.net/search/projects) — Cross-disciplinary design; useful for print, branding, and motion design aesthetics.

- **Google Images** — General fallback for broad aesthetic research.

These are examples. Use whatever image sources are accessible via your configured browser and search tools.
