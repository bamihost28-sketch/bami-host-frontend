**Palette**: Black fields, amber/orange highlights, yellow status, cyan links, green positive data, red exceptions, and grey separators (`#050505`, `#FF8A00`, `#FACC15`, `#00C8FF`, `#22C55E`, `#EF4444`).

**Type**: Mono-ish or condensed sans, uppercase mnemonics, tabular numerals, dense small labels, and command-line abbreviations. Use compact line heights and strong digit alignment.

**Texture**: Terminal flatness, ruled separators, compact chart blocks, menu ribbons, blinking cursors, tick tables, and low-resolution workbench density. Avoid glassy fintech minimalism.

**Shape**: Hard-edged windows, nested panels, horizontal ribbons, command boxes, table cells, and chart panes. Radius is zero or near-zero; panel hierarchy is rectangular.

**Motion**: Blinking cursors, live ticks, alert flashes, tiny chart refreshes, keyboard-focus jumps, and terse command feedback. Motion is rapid and data-driven (`80–180ms`).

**Spatial**: Maximal density, multi-panel tiling, secondary data everywhere, persistent command/status rows, and almost no decorative whitespace. The screen is a workstation, not a landing page.

**Cultural markers**: Ticker codes, keyboard legends, mnemonic commands, trader chat, market monitors, bond/equity tables, terminal help codes, orange/amber hardware memory.

**Non-negotiables**: black-and-amber finance terminal palette + command/workbench density + hard-edged paneling + live-tick/alert semantics. If these are removed, the result becomes a generic adjacent style rather than Bloomberg Terminal Monochrome.

## Connotation

**Authentic expert-workbench mode**. It signals financial throughput and learned operator culture; using only black/orange without command density becomes generic cyber dashboard.

## Scope

Suitable for finance dashboards, analytics workbenches, alert consoles, market games, and expert tools. Not suitable for consumer fintech onboarding or calm wealth-management content without substantial simplification.

**Related / Subsets**: Distinct from `high-performance-hmi`: finance-native command/workstation culture rather than industrial control doctrine. Distinct from `swiss-international`: throughput and compression over typographic objectivity.
