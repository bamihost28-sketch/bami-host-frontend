# References — Image Analysis

## Color Science & Extraction

- Fairchild, M. D. (2013). *Color Appearance Models* (3rd ed.). Wiley-IS&T. → Theoretical grounding for color role assignment and perceptual clustering.

- Itten, J. (1970). *The Elements of Color*. Van Nostrand Reinhold. → Color harmony systems relevant to palette cluster analysis.

- **CSS Color Level 4 specification** — [https://www.w3.org/TR/css-color-4/](https://www.w3.org/TR/css-color-4/) — Reference for color function syntax when generating output.

## Typography Reference

- Bringhurst, R. (2004). *The Elements of Typographic Style* (3rd ed.). Hartley & Marks. → Classification of type by historical period, style, and weight; basis for the "historical style" inference taxonomy.

- **Google Fonts Knowledge** — [https://fonts.google.com/knowledge](https://fonts.google.com/knowledge) — Practical guide to type classification and identification characteristics; useful reference for type category inferences.

- **MDN Web Docs: CSS `filter`** — [https://developer.mozilla.org/en-US/docs/Web/CSS/filter](https://developer.mozilla.org/en-US/docs/Web/CSS/filter)
- **MDN Web Docs: CSS gradients** — [https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_images/Using_CSS_gradients](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_images/Using_CSS_gradients)
- **MDN Web Docs: `box-shadow`** — [https://developer.mozilla.org/en-US/docs/Web/CSS/box-shadow](https://developer.mozilla.org/en-US/docs/Web/CSS/box-shadow)

## Accessibility

- **WCAG 2.2 Contrast Guidelines (SC 1.4.3)** — [https://www.w3.org/TR/WCAG22/#contrast-minimum](https://www.w3.org/TR/WCAG22/#contrast-minimum) — AA contrast ratio minimum (4.5:1 for normal text, 3:1 for large text) used in contrast pair notes.

- **WebAIM Contrast Checker** — [https://webaim.org/resources/contrastchecker/](https://webaim.org/resources/contrastchecker/) — Recommended tool for verifying extracted hex contrast pairs.

## Responsive Design & Layout

- **MDN Web Docs: CSS Grid Layout** — [https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_grid_layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_grid_layout) — Reference for grid-template-columns, grid-template-rows, and related properties used in layout extraction output.

- **MDN Web Docs: CSS Media Queries** — [https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_media_queries/Using_media_queries](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_media_queries/Using_media_queries) — Reference for breakpoint syntax used when documenting responsive behavior.

- **MDN Web Docs: CSS z-index** — [https://developer.mozilla.org/en-US/docs/Web/CSS/z-index](https://developer.mozilla.org/en-US/docs/Web/CSS/z-index) — Reference for z-axis layering values extracted from UI screenshots.
