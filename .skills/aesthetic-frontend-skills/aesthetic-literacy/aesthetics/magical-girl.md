**Palette**: Candy pink, moonlit white, lilac, sky blue, candy red, metallic gold/silver, and jewel accents. Use pastel grounds with saturated emblem colors (`#F9A8D4`, `#FFF7FB`, `#C4B5FD`, `#7DD3FC`, `#E11D48`, `#F6C453`) so transformation objects read as powerful rather than merely cute.

**Type**: Rounded cute sans for UI text, with emblematic display lettering for titles and badges. Pair Nunito/Quicksand-like softness with occasional high-contrast sparkle caps; use medium-bold labels, generous line-height, and badge text in small caps or arched lockups.

**Texture**: Gloss, glitter, jewel shine, laminated charm surfaces, prismatic overlays, and soft starfield grain. CSS can use radial gradients, conic glints, box-shadow halos, and small SVG sparkle masks; avoid generic flat pastel minimalism.

**Shape**: Hearts, stars, crescents, ribbons, oval gems, circular seals, scalloped cards, and compact mascot bubbles. Border radii are high (`18–32px`) except for crystalline gem facets and wand-like separators.

**Motion**: Radial reveals, spark bursts, orbiting particles, transformation wipes, charm unlocks, and confident state changes. Timing is bright and ceremonial (`220–600ms`, ease-out/back for spark accents), with reduced motion falling back to static emblem swaps.

**Spatial**: Medallion-centered compositions, framed status cards, character/team-color segmentation, collectible inventory grids, and circular hierarchy around a central transformation object. Whitespace is decorative framing rather than austerity.

**Cultural markers**: Wands, brooches, tiaras, mascots, sailor collars, school-to-heroine duality, moon phases, ribbons, guardian teams, collectible charms, friendship/empathy power language.

**Non-negotiables**: transformation iconography + jewel/gloss surface language + heart/star/moon emblem grammar + empowerment-through-ornament tone. If these are removed, the result becomes a generic adjacent style rather than Magical Girl.

## Connotation

**Nostalgic quotation or contemporary revival**. Apply the heroine-transformation grammar with enough modern spacing and accessibility to avoid costume parody; ironic pastiche should be reserved for explicitly camp projects.

## Scope

Suitable for achievement systems, identity/profile states, onboarding transformations, loyalty tiers, youth culture projects, collectibles, and celebratory dashboards. Not suitable for sober administrative workflows or contexts where sparkly heroism trivializes user needs.

**Related / Subsets**: Distinct from `decora-kei`, which is street-fashion overload; Magical Girl needs narrative transformation symbols and power-state hierarchy. Distinct from `y2k`, whose gloss is techno-futurist rather than heroine-emblematic. `fairy-kei` is softer thrift-pastel nostalgia without the transformation device grammar.
