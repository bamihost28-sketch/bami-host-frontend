**Palette**: nicotine beige, ABS off-white, olive, CRT green, warning amber.

**Type**: monospace, engraved labels, stencils, all-caps panel legends.

**Texture**: molded plastic, ribbed buttons, brushed metal, CRT bloom.

**Shape**: chunky boxes, bezels, toggles, rectangular modules, fasteners.

**Motion**: blink, scan sweep, coarse cursor steps, relay state changes.

**Spatial**: cockpit/control-panel density with bounded physical controls.

**Cultural markers**: Alien/Nostromo, repairability, used future, physical media.

**Non-negotiables**: nicotine beige, ABS off-white, olive, CRT green, warning amber + molded plastic, ribbed buttons, brushed metal, CRT bloom + chunky boxes, bezels, toggles, rectangular modules, fasteners + cockpit/control-panel density with bounded physical controls.

## Connotation

Cassette Futurism carries a specific robotics/speculative-technology connotation; use the source-backed cues above as a system rather than as isolated moodboard decoration.

## Scope

Use when the product, interface, or visual artifact benefits from this exact machine/body/worldbuilding language. Avoid when a neighboring aesthetic would communicate the intent with fewer cultural or accessibility risks.

## Related / Subsets

Adjacent to atomic-age but less optimistic; adjacent to cyberpunk but less urban/body-mod centered; distinct from high-performance HMI because it is diegetic retrofiction.
